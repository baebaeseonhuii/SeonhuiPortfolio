Tagging Out

This project is a quiz-style game application which uses AR to match the objects and text for English learners. By implementing various objects such as toys, airplanes, and chairs with AR technology, it adds a visual element to help users learn and memorize new vocabulary.

Development Environment

I developed this project using SwiftUI, ARKit and RealityKit API. I also created object models by using AR QuickView at Apple and Reality Composer.

App's Flowchart

As recognizing proper space, choose an AR object to see what it is like. After choosing the object, press the keyboard icon on the top and input the right word of the object.

Result 1

The image of the left side is result of generating AR objects. The image on the center shows keyboard for users to input texts by touching the keyboard icon on the top. The image on the right side shows the result of text that users type.

Result 2

If it is correct, there comes "Correct" and button which goes to the next quiz. Also, If it is not correct, there comes "Try again" and button which can delete the text.

Yeon-Haneul(Light Sky)

This project is a social networking app for connecting missing dogs with owners.

Logo & Theme

World Animal Day is the day on October, which is 4th of October and October means "Month of Sky" in Korean. Our team decided light sky to be a theme color of this project.

Wireframe

The application includes Home, Community, Volunteer, Chat, and Setting. Using Tab Bar Controller with navigation, set main tabs on the bottom.



Home

Main functions of "Home" are divided into 2 tabs which are "Finding Owner" and "Finding Dog". By posting, users can share pictures, type of dog, location, and feature of the dog. Uploaded posts are listed on TableView via Realtime Database on Firebase.



Community

"Community" functions for users to communicate with each other as a forum.



Volunteer

"Volunteer" offers users opportunities to volunteer at shelters. Users can choose pictures, location using PickerView and date using DatePicker. They also can apply for posted volunteer.



Setting

In "Setting", Users can edit their information such as profile and nickname, account and post history. Users can have access to their previous posts, using Storage on Firebase.


Result 1

"Sign In and Sign Up" is on the left side of images. As users touch one of TableView's cells, the right image is showed with modalPresentationStyle. In this page, users can contact writers by touching the button on the bottom.



Result 2

In "Community", Users can connect with each other regarding life, dogs, animals and so on. It offers search fuction through the magnifying glass button on the upper right corner of the screen using UISearchBar.



Result 3

The fisrt two images show "Chat" tab. It is used when users check with each other about posts or volunteers. The right side of the images shows confirmation alert as a user touch the "Join" button on the bottom.

Veggie Hero

It is a "Veggie Hero" role-playing game where you fight against garbage monsters to save the planet in a world overflowing with garbage due to excessive environmental pollution.

Characters

On the far left is "Veggie Hero", the main character of the game. The hero fights an enemy armed with trash on the center image. On the far right is a sick earth as an npc.



Attack

The three icons on the left are the weapons and items used by Veggie Hero. You can replenish your strength with the Recycle button, and you can effectively attack nearby villains with the broccoli icon. The icon on the far right is the trash bomb used by the villain.



Map

When you run the app, you can check the movement of the characters on the mini-map in the upper left corner of the screen that appears. You can control the movement with the action keys in the lower left corner. The upper center indicates the level, and you can attack with the items on the lower right.

Result

When you receive a mission from the npc and go to the portal, you will find a place where you fight a villain who throws garbage bombs. When you defeat the villain, a random globe-shaped reward appears.

Hello World Hello Woman

무중력지대 양천의 고용노동부 청년센터 커뮤니티 지원 사업으로 진행한 프로젝트의 결과물입니다. "Women TechMakers" 컨퍼런스에서 만나 스터디를 꾸린 4명의 테크업계 여성들과 각자의 관심사를 바탕으로 아카이브 목적의 웹 사이트를 제작하였습니다.

개발 환경

아톰 에디터를 사용하여 홈페이지를 제작하고 관리하였습니다. HTML5와 CSS3 그리고 Bootstrap을 활용하여 반응형 웹으로 구성하였습니다. 버전 관리는 깃허브를 활용하였습니다.

웹 페이지 구성 - 메인

프로필 카드로 메인 페이지를 구성하였습니다. 왼쪽으로 MENU가 있고 MENU를 누르면 세부 카테고리가 나타납니다.



웹 페이지 구성 - Music

카테고리 중 첫번째인 Music을 누르면 자동으로 음악이 재생되어 나오고 중앙에는 자주 듣는 사운드 클라우드 음악을 정사각형으로 배치하여 페이지를 구현하였습니다.



웹 페이지 구성 - Books
